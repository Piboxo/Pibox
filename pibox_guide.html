PiBox Setup Guide

Your PiBox is pre-configured — most settings are already done. This guide covers what you need to do yourself (accounts, router setup) and what's already in place.

✓ Already done on your PiBox BIOS optimized (virtualization ON, sleep disabled, auto power-on) · Windows high performance mode · Apps auto-start at boot · Process priorities set · Weekly reboot scheduled (Sunday 11pm) · Firewall configured
→ What you need to do Create accounts on each app (with your email) · Set up your router (fixed IP + UPnP) · Connect your payment methods on each platform
Step 1

Create your accounts

Each app needs an independent account. Your earnings go directly to your accounts — PiBox has no access to them.

Required
Sign up on each app

Open each app on your PiBox (they're on the desktop or taskbar). Create an account via Sign Up with your email. Once logged in, the app starts earning automatically.

  • Salad.io → salad.com
  • earn.app → earn.app
  • MystNodes → mystnodes.com
  • bitping → app.bitping.com
Step 2

ProtonMail + MetaMask wallet

Two one-time setups that protect your identity and enable crypto payouts.

Recommended
Create a ProtonMail email proton.me — free

Use a dedicated ProtonMail address for all your PiBox apps (e.g. yourname.pibox@proton.me). Gmail tracks activity and some platforms flag repeated automated logins as bot behavior — ProtonMail avoids this.

→ proton.me Free, end-to-end encrypted, no ads, no tracking. Takes 2 minutes to set up.
Required for crypto payouts
Create a MetaMask wallet metamask.io — free

MetaMask is the most widely used Web3 wallet. Install it as a browser extension (Chrome, Firefox, or Edge).

1. Go to metamask.io (check the URL carefully — fake sites exist).

2. Click "Download for Chrome" → Add Extension → Create a new wallet.

3. Critical: Write your 12-word seed phrase on paper. Store it somewhere safe. Never enter it online. Never share it with anyone — not even PiBox.

4. Your wallet address looks like: 0x1a2b...f9e8. This is what you paste in apps for crypto payouts.

⚠ Create a dedicated wallet for PiBox Do NOT use a wallet that holds significant funds. Create a fresh wallet specifically for PiBox earnings. Anyone with your seed phrase controls your wallet.
Optional
Add Polygon network to MetaMask — lower fees

Some apps pay on Polygon instead of Ethereum to save on transaction fees. Add it in MetaMask → Settings → Networks → Add Network:

Network Name: Polygon RPC URL: https://polygon-rpc.com Chain ID: 137 Symbol: MATIC
Step 3 — Network

Fixed local IP for your PiBox

Gives your PiBox a stable address on your home network. Apps work better without IP changes. Done once in your router.

Recommended
Reserve a static DHCP lease in your router

1. Find your PiBox MAC address — open Command Prompt (Win+R → cmd):

ipconfig /all // Note "Physical Address" of your Ethernet adapter // Format: XX-XX-XX-XX-XX-XX

2. In your router admin page → DHCP → Static leases → assign that MAC to a fixed IP (e.g. 192.168.1.50).

Common router IPs: Orange Livebox 192.168.1.1 · Free 192.168.1.254 · SFR 192.168.1.1 · Bouygues 192.168.1.254

Step 4 — Network

UPnP & port forwarding

MystNodes earns more as a "full cone NAT" node. UPnP handles this automatically.

Recommended
Enable UPnP on your router

In your router admin → look for UPnP or NAT/PAT → enable it. Most French routers have it enabled by default.

Optional
Manual port forward for MystNodes (if UPnP is off)

Forward port 4449 (TCP + UDP) to your PiBox fixed IP. In MystNodes → Settings → Node status: if you see "Full cone NAT", you're good. If "Symmetric NAT", enabling UPnP or forwarding the port will improve earnings.

Step 5 — Windows

Verify apps auto-start

Already configured. Just verify it's correct.

Pre-configured
Check startup apps

Open Task Manager: Ctrl+Shift+Esc → Startup tab. All PiBox apps should show Enabled. If any show "Disabled", right-click → Enable.

Should be Enabled: Salad · earn.app · MystNode Launcher · bitping
Step 6 — Windows

Weekly reboot — configured

A reboot is scheduled every Sunday at 11pm. It clears memory, applies Windows updates, and keeps apps healthy.

Pre-configured
View the scheduled task

Win+R → taskschd.msc → Library → look for PiBox_WeeklyReboot. You'll see Sunday 11:00pm.

Salad.io note
⚠ Consider disabling the reboot if Salad is earning well

Salad assigns compute "jobs" — some pay significantly more than others. If you reboot while on a high-paying job, Salad takes time to reassign a new one, and your node reputation score can temporarily drop.

Recommendation: If Salad is actively earning well, disable the weekly reboot and only restart when something seems buggy.

To disable: Task Scheduler → PiBox_WeeklyReboot → right-click → Disable
To re-enable: same steps → right-click → Enable
The task stays saved — you can toggle it anytime.
Good times to reboot: Apps not responding · Salad stuck on "Initializing" for 30+ min · Windows update pending · Once a month minimum.
Optional
Change the reboot time or day

Task Scheduler → double-click PiBox_WeeklyReboot → Triggers tab → double-click the trigger → change day/time → OK.

Optional
Recreate the task (if accidentally deleted)
# PowerShell as Administrator $action = New-ScheduledTaskAction -Execute "shutdown.exe" -Argument "/r /f /t 60" $trigger = New-ScheduledTaskTrigger -Weekly -DaysOfWeek Sunday -At "23:00" $settings = New-ScheduledTaskSettingsSet Register-ScheduledTask -TaskName "PiBox_WeeklyReboot" -Action $action -Trigger $trigger -Settings $settings -RunLevel Highest -Force
